Es necesario descargar SageMath y agregarlo al PATH para poder correr el programa.

Para correr el programa es necesario usar la terminal y ubicarse en donde está el archivo "mainMetNum2.py".
El comando para correr el programa es el siguiente:

sage -python3 mainMetNum2.py

o

sage -python mainMetNum2.py

Para ingresar las funciones al programa es necesario crear un archivo de texto con formato '.txt' y escribir las funciones como en los ejemplos "FunMul.txt" y "Funciones.txt".
En el archivo "Funciones.txt" se muestra un ejemplo de como se deben ingresar las expresiones para aplicar el método de Punto Fijo.
En el archivo "FunMul.txt" se muestra un ejemplo de como se deben ingresar las expresiones para aplicar el método de Newton y Broyden para sistemas de ecuaciones no lineales.
Los archivos de texto con las expresiones se deben guardar en la misma ruta que el archivo "mainMetNum2.py".

En caso de no saber como ingresar la expresión, puede consultar la documentación de SageMath.
